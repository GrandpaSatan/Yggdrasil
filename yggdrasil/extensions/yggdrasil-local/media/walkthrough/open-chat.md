# Open the Chat Panel

The chat panel streams from Odin in real time. You can:

- **Pick a model** from the dropdown (loaded models marked `●`)
- **Pin a flow** to use multi-step pipelines
- **Attach context** — current file or selection
- **Use slash commands**:
  - `/flow coding_swarm write a fibonacci function`
  - `/model nemotron-3-nano:4b explain this code`
  - `/memory rust async patterns`
  - `/help` for the full list

**What to do:**
1. Press `Ctrl+Shift+I` (or run `Yggdrasil: Open Chat` from the command palette)
2. Send a message — try `write a hello world in rust`
3. Watch tokens stream in real-time

**Other shortcuts:**
- Select code → right-click → `Yggdrasil: Explain Selection` (or `Ctrl+Shift+E`)
- Right-click → `Yggdrasil: Edit With Model` for guided edits

You're all set. Happy prompting.
