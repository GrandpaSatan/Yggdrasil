# Welcome to Yggdrasil

Yggdrasil is a self-hosted local-AI ecosystem — a model fleet (Ollama-backed), an intent-aware router (Odin), an engram memory (Mimir), and a Continue/Cline-style chat client right here in VS Code.

This walkthrough takes about a minute and gets you connected to your Yggdrasil server.

If you don't have one set up yet, see the [Yggdrasil README on GitHub](https://github.com/GrandpaSatan/Yggdrasil) for the quick self-host guide (Munin + Hugin nodes, native Ollama, Odin router).

After you finish:
- `Ctrl+Shift+I` opens the chat panel
- `Ctrl+Shift+Y` opens the Flows Explorer
- The Yggdrasil icon in the activity bar gives you the sidebar trees
